# vue3-project

这是一个基于vue3+elementPlus+echarts5+cesium 的项目, 主要体现图表和地图相关的功能

## 注意node 版本

node >= 18 

## Project Setup

```sh
yarn
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
